# OODS Foundry Cookbook

Practical recipes showing common OODS workflows using MCP tool calls.

| Recipe | Description |
|--------|-------------|
| [List to Detail](./01-list-to-detail.md) | Compose a list layout with behavioral traits, then a detail view for a selected item |
| [Dashboard with Charts](./02-dashboard-with-charts.md) | Compose bar + line visualizations into a dashboard layout |
| [Multi-Brand Theming](./03-multi-brand-theming.md) | Apply different brand configs and build themed token artifacts |
| [Full Pipeline](./04-full-pipeline.md) | Run the complete pipeline from object to saved schema to code generation |
| [Schema Iteration](./05-schema-iteration.md) | Save a schema, load it back, modify with compose overrides, and re-save |

Each recipe includes:
- Problem statement
- Step-by-step tool calls
- Expected output snippets
