# Runbooks

Fast paths for common OODS work. Each runbook is action-first and keeps steps short.

- [Add a Trait](./add-a-trait.md) - create a new capability trait and validate it.
- [Add an Object](./add-an-object.md) - compose traits into a canonical object.
- [Add a Chart](./add-a-chart.md) - ship a normalized spec and story quickly.
- [Change Tokens Safely](./change-tokens-safely.md) - update tokens with guardrails.
- [Update Status Map](./update-status-map.md) - adjust billing statuses and StatusChip.
